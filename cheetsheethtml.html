HTML Cheat Sheet

Document Structure

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>Title</title> </head> <body> Content here </body> </html>

Text

<h1>Heading 1</h1> <h2>Heading 2</h2> <h3>Heading 3</h3> <p>Paragraph</p> <br> Line break <hr> Horizontal line <strong>Bold</strong> <em>Italic</em> <u>Underline</u> <mark>Highlight</mark> <code>Code</code>

Lists

<ul> <li>Item 1</li> <li>Item 2</li> </ul> <ol> <li>Item 1</li> <li>Item 2</li> </ol>

Links & Media

<a href="url">Link</a> <a href="mailto:email">Email</a> <a href="tel:phone">Phone</a> <a href="#" target="_blank">New tab</a> <img src="image.jpg" alt="text">

Form Basics

<form action="/submit" method="post"> <label for="name">Name:</label> <input type="text" id="name" name="name"> <textarea name="message"></textarea> <button type="submit">Submit</button> </form>

Input Types

<input type="text"> <input type="email"> <input type="password">
<input type="number"> <input type="date"> <input type="file">
<input type="checkbox"> <input type="radio"> <input type="hidden">

Table

<table> <tr> <th>Header</th> <th>Header</th> </tr> <tr> <td>Data</td> <td>Data</td> </tr> </table>

Semantic Elements

<header></header> <nav></nav> <main></main> <article></article> <section></section> <aside></aside> <footer></footer> <div></div> <span></span>

id="unique" Unique identifier class="style" CSS class
style="css" Inline CSS title="tooltip" Hover text
