CSS Cheat Sheet

CSS Structure

selector { property: value; property: value; } /* Comment */

Selectors

element /* h1 */ .class /* .container */ #id /* #header */ * /* all elements */ element.class /* div.box */ element, element /* h1, h2 */ element element /* div p */ element > element /* div > p */ element + element /* h1 + p */ element:hover /* hover state */ element::before /* pseudo-element */

Text & Fonts

color: red; font-family: Arial, sans-serif; font-size: 16px; font-weight: bold; font-style: italic; text-align: center; text-decoration: underline; text-transform: uppercase; line-height: 1.5; letter-spacing: 2px; word-spacing: 5px;

Colors

color: red; color: #ff0000; color: rgb(255, 0, 0); color: rgba(255, 0, 0, 0.5); color: hsl(0, 100%, 50%); color: hsla(0, 100%, 50%, 0.5); background-color: blue; background: #333;

Box Model

width: 100px; height: 50px; padding: 10px; margin: 20px; border: 1px solid black; padding: 10px 20px; /* top/bottom left/right */ padding: 10px 15px 20px 25px; /* top right bottom left */ margin: auto; /* center */

Display & Position

display: block display: inline display: inline-block display: none
position: static position: relative position: absolute position: fixed
top: 10px; right: 20px; bottom: 30px; left: 40px; z-index: 999;

Flexbox

display: flex; flex-direction: row; /* column */ justify-content: center; /* flex-start, flex-end, space-between */ align-items: center; /* flex-start, flex-end, stretch */ flex-wrap: wrap; gap: 10px; /* Flex items */ flex: 1; flex-grow: 1; flex-shrink: 0; flex-basis: 200px;

Grid

display: grid; grid-template-columns: 1fr 2fr 1fr; grid-template-rows: 100px auto; grid-gap: 20px; grid-column: 1 / 3; grid-row: 1 / 2; /* Grid areas */ grid-template-areas: "header header" "sidebar main" "footer footer";

Border & Border-radius

border: 1px solid black border-top: 2px dashed red border-radius: 10px
border-width: 1px border-style: solid border-color: blue

Background

background-color: #333; background-image: url('image.jpg'); background-repeat: no-repeat; background-position: center; background-size: cover; background-attachment: fixed; /* Shorthand */ background: #333 url('img.jpg') no-repeat center;

Shadow & Effects

box-shadow: 2px 4px 8px rgba(0,0,0,0.3); text-shadow: 1px 1px 2px #000; opacity: 0.5; visibility: hidden; overflow: hidden; /* visible, scroll, auto */ cursor: pointer;

Transform & Animation

transform: translate(50px, 100px); transform: rotate(45deg); transform: scale(1.5); transform: skew(20deg, 10deg); transition: all 0.3s ease;
@keyframes slide { from { left: 0; } to { left: 100px; } } animation: slide 2s infinite;

Responsive Design

/* Media queries */ @media (max-width: 768px) { .container { width: 100%; } } @media (min-width: 1024px) { .grid { grid-template-columns: repeat(3, 1fr); } }

CSS Reset/Normalize
